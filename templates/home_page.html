This project analyses the dataset that was generated by web scrapring of the site https://gadgets.ndtv.com.
The data of all available phones on the site was extracted and cleaned in 4 steps. The links to the codes are here:
  • The web scraper
  • Cleaning Step 1
  • Cleaning Step 2
  • Cleaning Step 3
  • Cleaning Step 4

The project does the following:

  1. Add Data

    • Add the Phone       : Add detail of a mobile phone to the database
    • Reset the data      : Clear all user fed data and restore the database to the original form

  2. Analyse

    • Query               : Query the database
    • Predict             : Machine Learning based price prediction of phone with specs given
    • Graph               : Interactive statistical graphs derived from the data

  3. View Databases

    • Phones              : View all data of phones currently in Database
    • Query History       : History of pamaters used for query
    • Predicition History : History of parametrs used for prediction and the result
